Project

This game was designed for the HTML5 & JavaScript Game Programming module of the University of West of Scotland (UWS).

The aim of this course was to create a basic playable browser game. However, this game must include few gameplay mechanics in order to discover and understand how a javascript game is working.

As it was only a 2 months project, all student used a free library framework 'Phaser' to help him to design his game.

Being a great amateur of Rogue-Like game (as Rogue Legacy, The Binding of Isaac,...), I decided to design a Rogue-Like game for this project : Scogue, my scottish rogue ! .


Mechanics

In this part I will introduce the main gameplay mechanics.

HealthPoint : This features able to know if the player is alive or not.
It is based on 2 points, the maximum HealthPoint Pool and the current life (which obviously can't be more important than the maximum HealthPoint pool)

When the HealthPoint of the player reach 0, you lose the life and also all of your progression...
However, as the main design of the game is based on 'Rogue Like' you will be able to use the collected resources (coins, score) to upgrade your current stats for the next life ! Be carrefull to spend the maximum of your resources, because you will lose of all it by starting a new life.


Damage : This features correspond to the based amount of damage you will inflict to your opponents
This value will be changing in function of the weapon used, there are 4 differents weapons gameplay available (hammer, bow, axe, sword). The direction of you attack is depending of the player direction.

Armor : The player have an amount of armor. This armor is used to calculate a percentage of armor reduction (logarithmic function) which will be able to prevent the player against important damage.

Coins : Is a resource of the game using to buy new stuff providing more amor, damage, hp.. but also to acquire some spells (as double jump, dash..). You will gather it from monsters and interactive object of the game (as chest...)

Speed : Define the movement speed of the player, this can be modificated when the player is walking on some kind of ground (as for example ice, mud, ...)

Invulnerability : Is used to prevent the player and the monsters to take to much damage in a short time. The sprite will be blinking if the character is actually invulnerable.

Jump : Is based on 2 main points. Firstly the Height of the jump, in order to limit the direct access to some platform. And the Delay which fix the time between 2 jumps. It can be modificated with items

Score / Time elapsed : Both of this mechanics are working together. During your life, you will improve your score by killing your opponents. Whereas the time elapsed is just depending of the time played. When you are die, you will receive an amount of "Glory" depending of the score realised and the time elapsed (less you have played more it will be important).
The Glory is used as the resouces to upgrade your main characteristic (HealthPoint, Armor...), and it will be reset when you are starting a new life.

Console Commands

If you want are interested to do some trick/cheat in the game, you can use the few of the command available below.

To enter this commands go on the game and open the inspection tools of you browser (Ctrl+Shift+I, or right click -> Inspect). Then go on the 'Console Tab' and insert the commands and enjoy !


Stats

player.setHpMax(200)
Set the maximum HealthPoint pool of the player at 200

player.setHp(10)
Set the current Hp of the player at 10

player.setDamage(20)
Set the damage of the player at 20, it is corresponding to the damage you will inflict if you hit your oppponents. Moreover the current damage are depending of the weapon used.

player.setArmor(20)
Set the armor of the player at 20. The armor able to set your current armor reduction (this amount is inserted into a logarithmic function...)to reduce the received damage

player.setSpeed(300)
Set the movement speed of the player at 300

player.setCoins(222)
Set the current coins of the player at 222


More Stats

player.setJumpHeight()
Set the height of the jump

player.setJumpTime()
Set the delay between 2 jumps

player.setScore(99999)
Set the score of the player for this life

player.setTimeElapsed(1,20,5)
Set the time of the current life played at 1h20min05s


Other

player.setInvulnerabilityTime(2000)
Set the time (in miliseconds) of invulnerability. Invulnerability state is used to prevent you against damage during a short time

player.setInvulnerability(1000)
Set the player invulnerable during 1s (when the player is invulnerable the sprite is blinking)

player.setGravityAllowed(false)
Enable or disable the gravity on the player

player.respawn()
If you want to restart this life at the start position

player.setKeyboard("AZERTY")
Set the keyboard used by the player (by default "QWERTY") : "AZERTY" or "QWERTY"
